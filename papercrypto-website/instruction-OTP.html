HOW TO USE OTP

Simple instruction how does OTP work ? We have 6 options to choice:

  • 1. Print 6x8 char table
  • 2. Print coding table
  • 3. Print decoding table
  • 4. Encode message
  • 5. Decode message
  • 6. Test alphabet sample string


Let me describe this options


  • 1. Print 6x8 char table

    When we get first option, the program shows the table which contains 6x8 signs. Program use it for random PAD generator.

    Example random 6*8 table:

  • 2. Print coding table

    Second options shows coding table.

  • 3. Print decoding table

    Third option shows decoding table.

  • 4. Encode message

    If we want to encode any message we have to choice option number four. When we get this option, program shows prompt "Write your message to code:". And now we must write the message which we want to encode. For example of message: Hello, my name is Jarek!. After this, program shows prompt about encryption PAD. PAD - What is it? There is key to decoding your message. PAD has always the same quantity of sings what your message. For example of PAD: 2K0=JQ2R5R-S$3QSA-HTo@=.

    Message: Hello, my name is Jarek! - 24 signs      PAD: 2K0=JQ2R5R-S$3QSA-HTo@=. 24 signs

    We have 2 way to generate your PAD of message, either autogenerate of PAD (option "A" - contraction from Auto) or you can write your own PAD, but you must remember about - PAD's signs equal Message signs (option "O" - contraction from Own).
    In next step program start encode your message and after encoding shows:

    First line shows how many signs has your message,
    Second line shows how many signs has your PAD,
    Third line shows the message which we want encode - "Coded message is:",
    Fourth line shows PAD of message - "Coding pad is:",
    Fifth line shows your coded message - "Coded message is:",

    For example:

    "...
    Write your message to code: hello
    Do you want autogenerated pad or you want to write down your own pad?(A/O) A
    Started encrypting...
    5
    5
    Coded message is: HELLo
    Coding pad is: K=V2i
    Coded message is: RA6.W
    ..."

  • 5. Decode message

    If you want decode your message it is very easy to do. You have to choice option five. Now program shows prompt about starting decoding of message. We must write our encode message and PAD of message. After program shows message which we encode.

    For example:

    "...
    Starting decoding of message!!
    Please write your encrypted message: RA6.W
    Please write your pad: K=V2i
    Decoded message is: HELLo
    ..."

  • 6. Test alphabet sample string

    This option show you what kind of sings we use to encode and decode the message.

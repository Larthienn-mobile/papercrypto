Alphabet used has 48 characters:

ABCDEFGHiJKLMNoPQRSTUVWXYZ0123456789^?!.*+-/=@$:

If we have the message which has more words than just one word, before encode the message all the small characters we change on big characters, (example a = A, ..., z = Z), all spaces we change on slashs (example " " = "/") but special character and digits are the same.

Example:
Message: I have 2 cats. Have you?
Message before encode: I/HAVE/2/CATS./HAVE/YoU?
